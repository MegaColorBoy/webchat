# WebChat
A web-based instant messenger similar to Facebook Messenger and Whatsapp.

## Why built this?
Just trying to reverse engineer and have some fun coding it.

## Technologies
- PHP
- Javascript
- MySQL
- Slim Framework (For REST API)
- SASS
- HTML and CSS
